h3. Picard: A Micro-framework for NodeJS

p. Usage: install Node (http://nodejs.org/) and start the app with `node app.js`. There are no other dependencies. 

!http://bloglikepattern.com/images/picard.png!

p. Right now it only supports stupid simple GET requests. It does, however, support Haml thanks to creationix's haml-js project.
